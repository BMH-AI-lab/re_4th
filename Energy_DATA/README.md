# 신재생 에너지 에너지 현황


1. 해양 에너지 
    + 해양 에너지 이용현황
        +  국내 조력 발전 이용 현황
            + 시화호 조력 발전소 
                + 경기도에 위치한 발전소, 단류식 창조발전 방식을 적용
                + 매년 약 552 GWh의 전기를 생산

        +  국외 조력 발전 이용 현황
            + 프랑스
                + 랑스(Rance)조력발전소를 보유 
                    + 단조지식 발전소로 단류식 발전과 복류식 발전이 모두 가능

            + 러시아
                + Kislaya Guba 발전소를 보유
                    + 0.2 MW급, 1.5 MW급 발전기를 설치하여 총 1.7 MW급 발전소로 운영
            
            + 중국
                + 장시아(Jiangxia) 조력발전소
                + Jiangxia 조력발전소는 복류식 발전으로 6개의
양방향 벌브형 발전기가 설치되어 있음
